Exempt Clause

Exempt clauses are clauses that are implied by the Code or aligns with its spirit but not made explicit when forming the main text of the Code. The adoption of the Code automatically assumes that these exempt clauses are in effect, but one may wish to explicitly write them out.

BDFL Exempt

For projects that adopts the Benevolent Dictator For Life (BDFL) model, BDFL is exempt from all restrictions of the Code. For projects that does not explicitly reject the BDFL model, the owner of the code is assumed to have BDFL status.

Code-of-Conduct Exempt

The development team of projects which adopts the Code retains full right to reject any proposal of Code of Conduct change at will. Such rejections are exempt from Clause 2 and 3, which normally requires a justification based on technological reasons.
